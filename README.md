## portifolio

# Paleta de cores:
#005f73 - azul escuro;
#1a7f64 - verde;
#ee9b00 - Laranja;
#f4f4f9 - Cor do fundo claro;
#0a1128 - Cor do fundo escuro;
#ffffff - Cor branca do texto presente no fundo escuro.

# Fontes utilizadas

Arial;
Arial Black.

As cores azul e verde foram utilizadas por preferência do aluno, cores adicionais foram usadas como contrastes com as cores principais.
Os estilos de fontes foram selecionadas por serem comuns no ambiente corporativo, principalmente Arial, por ser considerada padrão em documentos que seguem as normas da ABNT.
